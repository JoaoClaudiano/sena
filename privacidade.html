Política de Privacidade

Última atualização: março de 2025

Este site foi criado com um único propósito: compartilhar fé, conhecimento e devoção a Santa Catarina de Sena. Não temos interesse comercial nos seus dados. Tratamos qualquer informação que você nos confie com o mesmo cuidado e respeito que Catarina tratava cada pessoa que a procurava.

1. Quem somos

O site Santa Catarina de Sena (joaoclaudiano.github.io/sena) é um projeto pessoal sem fins lucrativos, dedicado à difusão da vida, obras e espiritualidade de Santa Catarina de Sena — Virgem, Mística, Doutora da Igreja e Padroeira da Itália e da Europa.

Para questões relacionadas à privacidade, entre em contato conosco.

2. Quais dados coletamos

Nosso site coleta o mínimo possível. Veja o que pode ser registrado:

2.1 Dados de navegação (analíticos)

Utilizamos um sistema próprio baseado no Supabase para registrar visitas de forma anonimizada. Os dados coletados incluem:

  • Página visitada (ex.: "biografia", "obras")
  • Tipo de dispositivo (computador, tablet, celular)
  • Origem da visita (referrer — de onde você veio)
  • Um identificador de sessão anônimo (gerado localmente, sem vínculo com sua identidade)

Não coletamos seu endereço IP de forma explícita. Não coletamos seu nome, e-mail ou qualquer dado pessoal identificável por meio das análises de navegação.

2.2 Velas virtuais acesas no Oratório

Quando você acende uma vela virtual no Oratório, podemos registrar:

  • A coordenada geográfica aproximada (apenas se você conceder permissão de localização ao navegador)
  • A intenção de oração, se você optar por registrá-la

A localização é usada exclusivamente para criar o mapa de velas acesas ao redor do mundo, como testemunho comunitário de fé. Nenhuma intenção de oração é compartilhada publicamente — permanece armazenada de forma privada.

2.3 Dados armazenados no seu dispositivo

Usamos o armazenamento local do navegador (localStorage e sessionStorage) para:

  • Lembrar sua preferência de modo noturno (claro ou escuro)
  • Lembrar o tamanho de fonte escolhido nos textos
  • Contar velas acesas neste dispositivo (apenas localmente)
  • Manter o identificador anônimo de sessão

Esses dados ficam apenas no seu dispositivo e podem ser apagados a qualquer momento limpando os dados do seu navegador.

2.4 Formulário de contato

Caso utilize o formulário de contato, os dados enviados (nome, e-mail e mensagem) serão utilizados exclusivamente para responder à sua mensagem. Não compartilhamos esses dados com terceiros.

3. Como usamos os dados

Os dados coletados têm propósitos muito específicos e benignos:

  • Melhoria do site: entender quais páginas são mais visitadas para melhorar o conteúdo.
  • Mapa do Oratório: exibir a distribuição geográfica das velas acesas como sinal comunitário de devoção.
  • Preferências pessoais: lembrar suas configurações de interface para uma experiência mais confortável.

Não vendemos, alugamos nem compartilhamos seus dados com terceiros para fins comerciais, publicitários ou de qualquer outra natureza.

4. Serviços de terceiros

Utilizamos alguns serviços externos que podem processar dados de acordo com suas próprias políticas:

  • Supabase (política de privacidade): banco de dados em nuvem onde armazenamos dados anonimizados de visitas e velas. Os dados são armazenados em servidores seguros na União Europeia.
  • Google Fonts (política do Google): carregamos fontes tipográficas dos servidores do Google, o que pode envolver o registro do seu IP pelo Google.
  • OpenStreetMap/Leaflet (política da OSM): para os mapas interativos das páginas Siena e Oratório.
  • jsDelivr CDN: para carregar bibliotecas JavaScript (epubjs, supabase-js). Pode registrar acessos anônimos aos recursos servidos.
  • GitHub Pages: hospedagem do site. O GitHub pode registrar dados de acesso de acordo com sua própria política de privacidade.

5. Cookies

Este site não utiliza cookies de rastreamento (como cookies de publicidade ou cookies de redes sociais). Utilizamos apenas o armazenamento local do navegador (localStorage) para preferências funcionais, conforme descrito no item 2.3.

Os botões de compartilhamento (Facebook, Twitter/X, WhatsApp) são links simples — eles não carregam scripts dessas plataformas enquanto você estiver em nosso site. Cookies dessas plataformas só são definidos se você clicar nos links e acessar os respectivos sites.

6. Seus direitos

De acordo com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), você tem os seguintes direitos em relação aos seus dados pessoais:

  • Acesso: saber quais dados possuímos sobre você
  • Correção: solicitar a correção de dados incorretos
  • Exclusão: solicitar a remoção dos seus dados
  • Portabilidade: receber seus dados em formato legível por máquina
  • Informação: ser informado sobre o uso de seus dados
  • Revogação do consentimento: a qualquer tempo, sem ônus

Para exercer qualquer desses direitos, entre em contato conosco.

7. Segurança dos dados

Adotamos medidas técnicas adequadas para proteger os dados coletados, incluindo:

  • Comunicação via HTTPS em todo o site
  • Políticas de Row-Level Security (RLS) no Supabase
  • Uso de chave pública anônima do Supabase (segura para exposição no frontend)
  • Nenhum dado de cartão de crédito ou financeiro é coletado ou processado

8. Crianças e adolescentes

Este site é destinado ao público em geral e não coleta intencionalmente dados de menores de 13 anos. O conteúdo é de natureza religiosa, cultural e educativa, adequado para todas as idades.

9. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Quando o fizermos, atualizaremos a data de "Última atualização" no topo desta página. Recomendamos que você verifique esta página ocasionalmente para se manter informado.

10. Contato

Se você tiver dúvidas, preocupações ou solicitações relacionadas a esta política, entre em contato conosco. Responderemos com atenção e presteza.
